# Contributing to WP-CLI Models

Thank you for considering contributing to WP-CLI Models!

## Code Style

- Follow PSR-12 coding standards
- Use PHP 8.1+ features (enums, readonly properties, etc.)
- Add type declarations everywhere possible
- Write meaningful commit messages

## Pull Request Process

1. Ensure your code passes all tests: `composer check`
2. Update documentation if needed
3. Add tests for new features
4. Update CHANGELOG.md
5. Submit PR with clear description
